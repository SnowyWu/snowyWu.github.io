# 人事管理系统

# 项目介绍

本项目是基于 ruoyi-vue 框架开发的人事管理系统，旨在帮助企业高效地管理人力资源，提高工作效率，减少管理风险。

# github 地址

https://github.com/snowyWu

# 运行环境要求

# 1. JDK 1.8+

# 2. Maven 3.0+

# 3. Node.js 10.0+

# 4. MySQL 5.7+

# 5. redis
